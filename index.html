Feature Topics

DATA VERSUS INFORMATION

A set of symbols used to represent things, events, actions, amounts, and other entities that can be processed or otherwise altered manually or automatically is referred to as "DATA".

VARIABLES

Variables are symbolic names that have a value attached to them that can change. Variables can be thought of as containers that can hold constant values that can be used while the program is running or being performed.

ARITHMETIC

To instruct the computer on how to evaluate an arithmetic expression involving numeric data into a single value, arithmetic operators are utilized in the expression.
